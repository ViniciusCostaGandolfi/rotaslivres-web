Colabore com nossa pesquisa

Utilize nossa ferramenta para otimizar suas entregas e ajude-nos a validar nossos algoritmos. Totalmente gratuito para ONGs e projetos sem fins lucrativos.

É um projeto acadêmico?

Sim, este é um projeto de pesquisa da Universidade Estadual de Campinas (UNICAMP) e seu uso é voltado para validação científica e impacto social.

Qual o custo de uso?

Nenhum. Como se trata de um projeto de pesquisa da UNICAMP focado em impacto social, a plataforma é 100% gratuita para ONGs e instituições de caridade.
